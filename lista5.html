Programação em shell script

A linguagem de script shell, frequentemente referida como "shell script" ou simplesmente "bash script",
não foi criada por uma única pessoa. Ela evoluiu ao longo do tempo como parte dos sistemas Unix e Unix-like.
A linguagem de script shell é usada para automação de tarefas e interação com o sistema operacional por meio de comandos e scripts.
O Bash (Bourne Again Shell) é uma das implementações mais populares e amplamente usadas da linguagem shell.
O Bash foi criado por Brian Fox em 1989, como um substituto do shell Bourne original (sh) criado por Stephen Bourne.
O Bash estende e aprimora as funcionalidades do shell Bourne original e adiciona recursos poderosos de programação de script.
Portanto, embora Brian Fox tenha criado o Bash, a linguagem de script shell em si é uma evolução natural das linguagens de comando e script que existiam nos sistemas Unix desde o seu início. Ela foi aprimorada e estendida por várias pessoas ao longo dos anos para se tornar a linguagem de script shell que conhecemos hoje. A programação em shell script é uma forma de automatizar tarefas em sistemas Unix-like (Linux, macOS, etc.) usando comandos de shell e scripts.

Exemplo1: Calculadora Simples

Este é um script que pede ao usuário para inserir dois números e realiza operações matemáticas simples.


#!/bin/bash

echo "Calculadora Simples"
echo "Digite o primeiro número:"
read num1
echo "Digite o segundo número:"
read num2
echo "Escolha a operação:"
echo "1. Adição"
echo "2. Subtração"
echo "3. Multiplicação"
echo "4. Divisão"
read operacao
case $operacao in
1)
resultado=$((num1 + num2))
;;
2)
resultado=$((num1 - num2))
;;
3)
resultado=$((num1 * num2))
;;
4)
resultado=$(echo "scale=2; $num1 / $num2" | bc) ;;
*)
echo "Opção inválida"
exit 1
;;
esac
echo "Resultado: $resultado"
